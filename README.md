# California Housing Price Prediction

This project predicts house prices using the California Housing dataset.

## Project Workflow
• Data Loading
• Data Cleaning
• Exploratory Data Analysis (EDA)
• Train-Test Split
• Linear Regression Model
• Model Evaluation

## Model Used
Linear Regression

## Evaluation Metrics
R² Score ≈ 0.57

## Key Insight
Median Income is the strongest predictor of housing prices.
